# 🚗 babyStickDrive

A toddler-friendly car game — collect ⭐ stars, beep 📣, and paint 🌈 rainbows!

## ▶️ Play Now

👉 **[Click here to play the game](https://bishumoharana.github.io/babyStickDrive/toddler_car_game_single_file.html)**

## Controls
- 🕹️ D-pad / Arrow keys / WASD — drive
- Space / H — beep
- X — paint rainbows
- C — celebrate 🎉
- P — pause

Works on mobile (touch), keyboard, and gamepad!
